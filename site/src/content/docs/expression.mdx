In Shasta, an expression is any of the following:

- A [literal value]
- A [variable]
- A [function]
- A [function call]

The following are all valid expressions:

<CodeBlock language="shasta">
  {`
"Hello, world!"
105
true
null
(+ 1 2)
{(printf "Hello, %s!" [$])}
`}
</CodeBlock>
